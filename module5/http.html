HTTP Protocol

Terms:

HTTP: Hyper Text Transfer Protocol
A request and response stateless protocol.
URN: Uniform Resource Name
Uniquely identifies the resource or name of resource. Does not include information on resource location.
URI: Uniform Resource Identifier
Uniquely identifies resource or resource location. Still may not provide info on how to get to resource. Example: /kuma/index.html is the path, but the server name is missing
URL: Uniform Resource Locator
Uniquely identifies resource location. Provides both the server name and the path to the resource. Example: http://www.mysite.com/kuma/index.html

GET /index.html?firstName=Maya&lastName=Name http/1.1

Reason no server is specified w/ GET is that the connection w/ the server must be established first

GET retrieves data from sever

POST send data to sever
